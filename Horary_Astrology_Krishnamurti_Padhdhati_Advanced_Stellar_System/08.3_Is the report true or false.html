Is the report true or false

One is to judge the third house for any information or correspondence or communication or report.

 

We have to find out the sub-lord of the third cusp; note in which constellation the sub-lord is deposited. If the lord of the constellation in which the sub-lord is deposited happen to be Saturn, the report is false; But if it is Mars then it is mischievous and false. If it is Jupiter, it is true; and if the lord happens to be any planet connected with Saturn in any manner, the report is false; but connected with Jupiter, the report will be correct. This is the way to find out whether a rumour is believable or not.

 

According to the Western system, note what the matter is. Whom does it refer? Find out the house which signifies the matter or the person. If the lord of that house is in an angular sign or if it be aspected by benefics the report is true. If any planet is deposited in that house which signifies the matter and receives favourable aspect from benefics, the report is true. Otherwise it is false.
